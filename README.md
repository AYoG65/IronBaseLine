# IronBaseline 🔐

IronBaseline is a cross-platform CIS Level 1 baseline hardening framework for
Windows Workstations, Windows Servers, Ubuntu, and RHEL.

## Features
- Audit, Harden, Rollback modes
- CIS Level 1 inspired controls
- Windows + Linux support
- Safe for production environments

## Supported Platforms
- Windows 10 / 11
- Windows Server 2019 / 2022
- Ubuntu 20.04+
- RHEL 8+
